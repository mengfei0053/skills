# OpenHarmony 6.0 Documentation Map

This file provides a quick reference to the major documentation sections available in the OpenHarmony 6.0 application development documentation.

## Core Framework Documentation

### Application Framework
- **Ability Kit** - `application-models/` - Application framework and lifecycle management
- **ArkData** - `database/` - Data management and persistence
- **ArkTS** - `arkts-utils/` - ArkTS language features and utilities
- **ArkUI** - `ui/` - User interface framework and components
- **ArkWeb** - `web/` - Web capabilities and WebView
- **Background Tasks Kit** - `task-management/` - Background task scheduling
- **Core File Kit** - `file-management/` - File system operations
- **Form Kit** - `form/` - Widget (card) development
- **IME Kit** - `inputmethod/` - Input method framework
- **IPC Kit** - `ipc/` - Inter-process communication
- **Localization Kit** - `internationalization/` - I18n and L10n

### System Services

#### Security
- **Access Token** - `security/AccessToken/` - Permission management
- **Asset Store Kit** - `security/AssetStoreKit/` - Secure asset storage
- **Crypto Architecture Kit** - `security/CryptoArchitectureKit/` - Cryptography
- **Data Protection Kit** - `security/DataProtectionKit/` - Data encryption
- **Device Certificate Kit** - `security/DeviceCertificateKit/` - Device certificates
- **Universal Keystore Kit** - `security/UniversalKeystoreKit/` - Key management
- **User Authentication Kit** - `security/UserAuthenticationKit/` - Biometric auth

#### Network & Connectivity
- **Connectivity Kit** - `connectivity/` - Bluetooth, NFC, WiFi
- **Distributed Service Kit** - `distributedservice/` - Distributed capabilities
- **Network Kit** - `network/` - HTTP, WebSocket, network management
- **Telephony Kit** - `telephony/` - Cellular communication

#### Basic Services
- **Basics Service Kit** - `basic-services/` - Common events, USB, print, etc.
- **FFRT Kit** - `ffrt/` - Function Flow Runtime (task scheduling)
- **Input Kit** - `device/input/` - Input device handling
- **MDM Kit** - `mdm/` - Enterprise device management

#### Hardware
- **Driver Development Kit** - `device/driver/` - Driver development
- **Multimodal Awareness Kit** - `device/stationary/` - Multi-sensor fusion
- **Sensor Service Kit** - `device/sensor/` - Sensor access

#### Debug & Optimize
- **Performance Analysis Kit** - `dfx/` - Logging, tracing, profiling
- **Test Kit** - `application-test/` - Application testing
- **Debugging Tools** - `tools/` - Development and debug tools

### Media Services
- **Audio Kit** - `media/audio/` - Audio playback and recording
- **AVCodec Kit** - `media/avcodec/` - Audio/video encoding and decoding
- **AVSession Kit** - `media/avsession/` - Media session management
- **Camera Kit** - `media/camera/` - Camera capture
- **DRM Kit** - `media/drm/` - Digital rights management
- **Image Kit** - `media/image/` - Image processing
- **Media Kit** - `media/media/` - Media player
- **Media Library Kit** - `media/medialibrary/` - Media file management

### Graphics
- **ArkGraphics 2D** - `graphics/` - 2D graphics rendering
- **ArkGraphics 3D** - `graphics3d/` - 3D graphics and rendering

### Application Services
- **Ads Kit** - `ads-service/` - Advertising services
- **Calendar Kit** - `calendarmanager/` - Calendar management
- **Contacts Kit** - `contacts/` - Contact management
- **Location Kit** - `device/location/` - Location services
- **Notification Kit** - `notification/` - Notification management

### AI
- **MindSpore Lite Kit** - `ai/mindspore/` - AI inference framework
- **Neural Network Runtime Kit** - `ai/nnrt/` - Neural network runtime
